<h1>How to play:</h1>
<ul>
<li>Move - WASD</li>
<li>Dash - Space</li>
<li>Basic attack - Left mouse click</li>
<li>Spin attack - Q</li>
<li>Dash attack - F</li>
<li>Esc - Back to main menu</li>
<li>Tab - How to play menu</li></ul>
